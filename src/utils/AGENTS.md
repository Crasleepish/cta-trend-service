# src/utils

## 作用与架构定位

* 职责：跨模块复用的纯函数工具（对齐、收益、指数重构等）。
* DOT 位置：辅助模块（不直接进入服务分层）。

## 目录结构（至多两层）

```
.
|-- __init__.py
|-- series.py
`-- trading_calendar.py
```

## 文件说明

* `series.py`：序列对齐、对数收益、指数重构的纯函数工具。
* `trading_calendar.py`：交易日历工具（区间、偏移、周末采样）。
* `__init__.py`：保持空，不放逻辑。

## 本模块约束/规范

* 纯函数、无副作用、禁止 DB/IO 依赖。
